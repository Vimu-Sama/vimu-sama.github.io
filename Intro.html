Scalable APIs • Secure Systems • Microservices

I am a Computer Science Engineering graduate from Jaypee Institute of Information Technology, Noida (2022). I currently work as a Backend Developer, focusing on designing and building secure, scalable, and production-ready backend systems.

My primary expertise lies in the Java ecosystem, where I work extensively with Spring Boot to build RESTful APIs, implement authentication and authorization, and structure applications following clean architecture and domain-driven principles.

I have hands-on experience working with relational databases like PostgreSQL, using Spring Data JPA for persistence, and optimizing performance through caching mechanisms such as Redis.

I actively work on microservices-based architectures, implementing service-to-service communication, API gateways, and stateless authentication using JWT. I also have experience with Kafka for event-driven systems and gRPC for efficient inter-service communication.

My focus as an engineer is on writing clean, maintainable code, understanding system design trade-offs, and building backend systems that can scale reliably in real-world production environments.
